Registration Numbers

Enter registration numbers from the following towns:

  • Cape Town(CA)
  • KuilsRiver(CF)
  • Paarl(CJ)
  • Bellville(CY)
  • Use either one of these formats e.g CF 218211/CA 222-222/CJ 555 555

Click "ADD" when you have entered the registration number in the text area.You should be able to filter the registrations by a town using the dropdown menu and choosing the town for which you want to view the registration numbers.
